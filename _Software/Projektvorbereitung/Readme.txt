Das Script CadDev_AEC_Aufbereiten-Super hilft beim zurücksetzen der Architekturzeichnungen auf eine gewünschte Farbe.
Die Scriptdatei kann in AutoCAD-MEP geladen werden.
Sie verwendet einen Befehl aus den CadDev-ACD.Utils http://www.cad-development.de/index.php?id=400
Diese sollten also installiert sein, um den vollen Umfang nutzen zu können.
